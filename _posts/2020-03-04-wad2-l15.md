## HTML vs XHTML

![](/static/2020-03-29-16-46-35.png)
![](/static/2020-03-29-16-50-35.png)

🍊 HTML

* html标签起始
* 标签大小写敏感
* 支持空标签，可以无闭标签
  * 如`<p>`
  * 如`<hr>`
* **不强制缩进**
* 属性可以无引号
  * `size=20`
* 支持属性最小化
  * `<table border>`
* 无严格标签布局约束
* 样式标签
  * `<font>`

🍊 XHTML

* 以XML声明起始
  1. `<!DOCTYPE	html>`
  2. `<html>`

* **标签全部小写**
* **标签必须闭合**
* **空标签必须闭合**
  * `<hr/>`
* **严格缩进**
* **属性必须引号引用**
* **不支持最小化属性**
  * `<table border="true">`
* 严格标签布局
  * p标签中不能嵌套h1
* **样式标签废弃，使用样式表**

### XML Differs from HTML

🍊 不同目的： carring vs displaying
![](/static/2020-03-29-16-55-47.png)

* XML 用于传输 & 存储数据
* HTML用于渲染数据

## JSON

### Definition

🍊 java script object notation

![](/static/2020-03-29-16-58-00.png)

* 轻量级数据交换格式
  * 可读性，可写性高
  * **易解析**&生成
  * **有效提高网络传输效率**

* 基于两种通用数据
  * **键值对集合**
    * 通常看作为，对象，数组，构建，字典，哈希对
  * **有序**值序列
    * 通常由数组，列表，向量实现

* 独立语言
